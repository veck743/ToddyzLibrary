Toddyz Library

Create Roblox interfaces easily.

Installation

Use the following code:

local Library = loadstring(game:HttpGet("COLOQUE_O_LINK_DA_LIBRARY_AQUI"))()

local Window = Library:CreateWindow({
    Name = "Meu Hub"
})

Example

local Tab = Window:CreateTab("Main")

Tab:CreateButton({
    Name = "Hello",
    Callback = function()
        print("Hello World!")
    end
})
