Toddyz Library

Biblioteca Lua para criação de interfaces no Roblox.

📦 Instalação

Carregue a biblioteca usando:

local ToddyzLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/veck743/ToddyzLibrary/main/Library.lua"))()

CreateWindow

local Window = ToddyzLibrary:CreateWindow("Meu Menu")

CreateTab

local Tab = Window:CreateTab("Principal")

CreateLabel

Tab:CreateLabel("Texto Informativo")

CreateButton

Tab:CreateButton("Executar", function()

print("Executado")

end)

CreateToggle

Tab:CreateToggle("Sistema", function(Estado)

print(Estado)

end)

CreateInput

Tab:CreateInput("Digite algo", function(Texto)

print(Texto)

end)

Notify

ToddyzLibrary:Notify("Olá!",3)

Recursos

Sistema de janelas com animações.
Abas com rolagem horizontal.
Botões, toggles e inputs personalizados.
Sistema de notificações.
Suporte para dispositivos mobile.
Interface arrastável e minimizável.
