Toddyz Library

Create beautiful Roblox interfaces easily.

Installation

Copy the code below into your Roblox script.

local Library = loadstring(game:HttpGet("COLOQUE_SEU_LINK_AQUI"))() local Window = Library:CreateWindow({ Name = "Meu Hub" })

Documentation

Use the Library object to create windows, tabs, buttons, toggles, sliders and more.

Example:

local Tab = Window:CreateTab("Main") Tab:CreateButton({ Name = "Hello", Callback = function() print("Hello World!") end })
