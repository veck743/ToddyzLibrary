Toddyz Library

Uma biblioteca simples, rápida e moderna para criar interfaces no Roblox.

📦 Instalação

Copie o código abaixo para começar.


local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/veck743/ToddyzLibrary/main/Library.lua"))()

local Window = Library:CreateWindow({
    Name = "Meu Hub"
})

🚀 Exemplo

local Main = Window:CreateTab("Principal")

Main:Button({
    Text = "Olá",
    Callback = function()

        print("Olá Mundo!")

    end
})

Main:Toggle({
    Text = "Auto Farm",
    Default = false,
    Callback = function(Value)

    end
})

Main:Input({
    Text = "Nome",
    Placeholder = "Digite aqui...",
    Callback = function(Text)

    end
})

✨ Componentes

🔘 Button

Cria um botão clicável.

✅ Toggle

Cria um botão de ligar e desligar.

⌨️ Input

Permite ao usuário digitar um texto.
